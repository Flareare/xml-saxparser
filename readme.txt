Программа для парсинга XML-файлов методом SAX.
Задание находится в файле task.docx.
XML-файлы находятся в каталоге data
Создал свои таблицы, т.к. бэкап не открывался (и скорее всего, не соответствовал бы действитльности) 
Так, например, поле zone в задании имеет числовой тип, хотя в XML несколько раз попадаются строковые 
значения (либо это невалидные данные, либо неточности в задании). Так же поле uuid в задании почему-то
не являлось уникальным.
Скрипты таблицы находятся в папке scripts в txt-формате. Необходимо их выполнить в любой пустой БД.
Выбор файла и подключение к БД осуществляются через консоль (1 файл за 1 выполнение программы).
